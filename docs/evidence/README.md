# Evidências de Execução

Esta pasta armazena evidências de conclusão de cada fase, conforme
exigido pela `docs/quality/definition-of-done.md`: capturas de tela,
saídas de execução de testes, relatórios de cobertura, relatórios de
acessibilidade e outros artefatos que comprovem que os critérios de
aceite de uma fase foram atendidos.

Convenção de organização sugerida:

```text
docs/evidence/
  fase-01-fundacao/
  fase-02-autenticacao/
  fase-03-perfil/
  ...
```

Nenhuma evidência foi produzida ainda — esta pasta é criada vazia na
Fase 0 e será populada a partir da Fase 1.
